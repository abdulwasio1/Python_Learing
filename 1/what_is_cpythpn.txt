## Sabse pehle ek baat pakki karo

### ❗ Python ≠ CPython

* **Python** = language (rules)
* **CPython** = program jo Python chalata hai
  (yeh program **C language** mein likha hua hai)

Tum jab “python file.py” chalate ho →
tum **CPython program** chala rahe hote ho.

---

## Ab ek simple scene imagine karo 🎬

### Tumhari Python file:

```python
print("Hello")
```

### Jab tum run karte ho:

```bash
python file.py
```

Actual mein kya hota hai?

---

## Step-by-step (slow, simple)

### Step 1️⃣

Tumhara `.py` code **C mein convert nahi hota** ❌

Yeh galat soch hai.

---

### Step 2️⃣

CPython tumhara code **bytecode** mein convert karta hai
(yeh chhoti-chhoti instructions hoti hain)

```
.py → bytecode
```

---

### Step 3️⃣

Ab **CPython ka engine** kaam shuru karta hai

Aur yeh engine:

* **C language** mein likha hota hai
* bytecode ko **read karta hai**
* aur **execute karta hai**

👉 Matlab:

> **C program Python instructions ko chala raha hota hai**

---

## Iska matlab ek line mein

> **Python code → CPython (C program) → CPU**

Tumhara Python code:

* Python hi rehta hai
* C nahi banta

Lekin:

* jo “machine” usko chala rahi hoti hai
* wo **C mein likhi hoti hai**

---

## Bohat strong analogy (yaad reh jayegi)

### Socho:

* Python code = **Urdu language**
* CPython = **Translator**
* Translator **English** janta hai (C language)

Tum Urdu bolte ho
Translator English mein samajh ke kaam karta hai

Urdu → English **nahi banti**
Lekin English wala aadmi **Urdu ka kaam kar deta hai**
